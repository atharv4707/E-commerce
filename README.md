# LUXORA - Ecommerce Website

A modern ecommerce website built with HTML, CSS, and JavaScript.

## Pages

- **index.html** - Homepage
- **products.html** - Product listing with filters
- **product-details.html** - Single product view
- **cart.html** - Shopping cart

## Features

- Browse products by category
- Filter and sort products
- Add to cart
- Responsive design

## How to Run

Just open `index.html` in a browser. No setup needed.

## Tech Used

HTML, CSS, JavaScript
